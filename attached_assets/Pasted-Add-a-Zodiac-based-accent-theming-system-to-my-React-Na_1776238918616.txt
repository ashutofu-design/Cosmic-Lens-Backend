Add a Zodiac-based accent theming system to my React Native (Expo) app.

Goal:
Personalize the app UI based on user's zodiac sign without breaking the design or making it look messy.

----------------------------------------

1. Detect Zodiac Sign:
- Use user's birth details (date of birth)
- Determine zodiac sign (Aries, Taurus, Gemini, etc.)

----------------------------------------

2. Do NOT change full theme:
- Do NOT change full background color
- Do NOT make the entire app red/green/etc.

Instead:
- Keep base theme (light/dark) same
- Only apply accent colors

----------------------------------------

3. Accent Color Mapping:

Aries → Red (#EF4444)  
Taurus → Green (#22C55E)  
Gemini → Mint/Light Green (#34D399)  
Cancer → Silver/Soft Blue (#93C5FD)  
Leo → Gold/Orange (#F59E0B)  
Virgo → Olive Green (#84CC16)  
Libra → Pink (#F472B6)  
Scorpio → Deep Red (#991B1B)  
Sagittarius → Purple (#8B5CF6)  
Capricorn → Dark Gray (#374151)  
Aquarius → Electric Blue (#3B82F6)  
Pisces → Aqua Blue (#06B6D4)

----------------------------------------

4. Where to apply accent colors:

- Buttons (primary actions)
- Active tabs / selected states
- Icons (highlighted ones)
- Borders (focus or selected)
- Small glow effects (subtle)

----------------------------------------

5. Where NOT to apply:

- Full background
- Full cards
- Large sections

----------------------------------------

6. Dynamic Theme System:

- Combine:
  Base Theme (Light/Dark) + Zodiac Accent

Example:
- Dark + Leo → dark UI + gold accents
- Light + Gemini → light UI + green accents

----------------------------------------

7. Smooth UI:

- Keep colors soft and premium
- Avoid neon or harsh colors
- Maintain consistency across screens

----------------------------------------

8. Fallback:

- If zodiac not available → use default accent (purple or blue)

----------------------------------------

Final Result:
- App looks personalized for each user
- UI remains clean and premium
- Users feel "this app is made for me"
