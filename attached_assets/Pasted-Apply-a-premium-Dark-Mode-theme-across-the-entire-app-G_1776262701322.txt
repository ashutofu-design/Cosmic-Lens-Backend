Apply a premium Dark Mode theme across the entire app.

Goal:
- High contrast
- Eye-friendly dark UI
- Luxury + modern feel
- No faded or low-visibility elements

----------------------------------------

1. BACKGROUND:

- Do NOT use pure black (#000000)
- Use deep dark navy:
  #0B0F19

----------------------------------------

2. CARDS & CONTAINERS:

- Background: #111827
- Border radius: 16
- Add subtle shadow or elevation

----------------------------------------

3. TEXT SYSTEM:

- Primary text: #F9FAFB (almost white)
- Secondary text: #D1D5DB
- Subtext: #9CA3AF

- Avoid low opacity or faded grey text

----------------------------------------

4. PRIMARY UI COLOR:

- Use Purple: #8B5CF6

Apply for:
- Selection states
- Tabs
- Icons (active)
- Input focus borders

----------------------------------------

5. CTA / BUTTON COLOR:

- Use Gold/Orange: #F59E0B

Apply for:
- Primary buttons (Save, Continue, Submit)

Optional gradient:
#F59E0B → #EA580C

----------------------------------------

6. INPUT FIELDS:

- Background: #1F2937
- Border: #374151

On focus:
- Border: #8B5CF6
- Add subtle glow

----------------------------------------

7. SELECTION STATES:

- Background:
  rgba(139,92,246,0.15)

- Border:
  #8B5CF6

----------------------------------------

8. WARNING / ALERT:

- Background: #451A03
- Text: #FCD34D

----------------------------------------

9. SUCCESS:

- Background: #052E16
- Text: #22C55E

----------------------------------------

10. BOTTOM TAB BAR:

- Background: #111827
- Top border: #1F2937

- Active tab:
  color: #F59E0B

- Inactive tab:
  color: #9CA3AF

----------------------------------------

11. ICONS:

- Default: #9CA3AF
- Active: #8B5CF6 or #F59E0B

----------------------------------------

12. GLOBAL RULES:

- No pure black UI
- No neon colors
- Maintain high contrast
- Use only 2–3 main colors
- Ensure readability in all screens

----------------------------------------

FINAL RESULT:

- Premium dark UI
- Clean, readable, and modern
- Luxury feel with gold + purple accents
