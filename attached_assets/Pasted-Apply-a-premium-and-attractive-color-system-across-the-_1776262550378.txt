Apply a premium and attractive color system across the entire app (Light Mode focused).

Goal:
- High visibility
- Strong contrast
- Premium modern look
- No faded or weak UI

----------------------------------------

1. BACKGROUND:

- Use soft white instead of pure white:
  #F8FAFC

- Avoid #FFFFFF as full screen background

----------------------------------------

2. CARDS & CONTAINERS:

- Background: #FFFFFF
- Border radius: 16
- Shadow: rgba(0,0,0,0.08)

- Do NOT use heavy borders

----------------------------------------

3. TEXT SYSTEM:

- Primary text (main content):
  #0F172A (dark navy, strong visibility)

- Secondary text:
  #334155

- Subtext / helper:
  #64748B

- Avoid light grey or faded text

----------------------------------------

4. PRIMARY COLOR (UI / INTERACTION):

- Use Purple: #6366F1

Apply for:
- Selected states
- Tabs
- Icons (active)
- Input focus border

----------------------------------------

5. CTA / BUTTON COLOR:

- Use Orange: #FF7A00

Apply for:
- Primary buttons (Save, Continue, Submit)
- Important actions only

Optional gradient:
#FF7A00 → #FF3D00

----------------------------------------

6. INPUT FIELDS:

- Background: #F1F5F9
- Border: #CBD5E1

On focus:
- Border: #6366F1
- Slight glow effect

----------------------------------------

7. SELECTION STATES (Buttons, Toggles):

- Background:
  rgba(99,102,241,0.06)

- Border:
  #6366F1

----------------------------------------

8. ALERT / WARNING:

- Background: #FEF3C7
- Border: #F59E0B
- Text: #92400E

----------------------------------------

9. SUCCESS:

- Background: #DCFCE7
- Text: #166534

----------------------------------------

10. BOTTOM TAB BAR:

- Background: #FFFFFF
- Top border: #E2E8F0

- Active tab:
  color: #FF7A00
  background: light orange tint

- Inactive tab:
  color: #64748B

----------------------------------------

11. ICONS:

- Default: #64748B
- Active: #6366F1 or #FF7A00

----------------------------------------

12. GLOBAL RULES:

- Do NOT use random colors
- Keep max 2–3 main colors only
- Maintain strong contrast
- Avoid washed-out UI

----------------------------------------

FINAL RESULT:

- Clean, bold, premium app UI
- Highly readable text
- Strong visual hierarchy
- Attractive modern design
