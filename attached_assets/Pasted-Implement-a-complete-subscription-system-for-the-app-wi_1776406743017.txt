Implement a complete subscription system for the app with two plans: BASIC and PRO, along with a 7-day free trial for new users.

----------------------------------------

GOAL:

- Clear difference between Basic and Pro
- High conversion through feature locking
- Smooth user journey (Free → Basic → Pro)

----------------------------------------

PLANS STRUCTURE:

1. FREE TRIAL (First-time users only)

- Duration: 7 days
- Access: ONLY BASIC features unlocked
- Ask Astro Expert: 3 questions per day
- No Pro access

Purpose:
Allow users to experience the app and build trust

----------------------------------------

2. BASIC PLAN — ₹199/month

Features:

- Ask Astro Expert: 10 questions per day

- Marriage Compatibility → BASIC version
- Love Compatibility → BASIC version
- Career → BASIC analysis
- Health → BASIC analysis
- Finance → BASIC analysis

- Future Timeline:
  → Only 1 month prediction unlocked

BASIC VERSION RULES:

- Show score + short summary
- Limited explanation
- No deep reasoning
- No karmic or advanced astrology insights
- Keep some curiosity for upgrade

----------------------------------------

3. PRO PLAN — ₹399/month

Features:

- Ask Astro Expert: Unlimited questions

- Marriage Compatibility → FULL PRO
- Love Reality → FULL analysis
- Career → Deep analysis
- Health → Deep analysis
- Finance → Deep analysis

- Future Timeline:
  → Full 6 months unlocked

- Include:
  - D1 + D9 analysis
  - Dasha (MD, AD, PD)
  - Karmic patterns
  - Hidden insights
  - Detailed reasoning

----------------------------------------

FEATURE LOCKING LOGIC:

Each section must have TWO levels:

- BASIC VERSION
- PRO VERSION

Example:

Marriage Compatibility:
- Basic → Score + short summary
- Pro → Full breakdown + reasons + deep analysis

Career:
- Basic → Overview
- Pro → Detailed prediction + timing + logic

Future Timeline:
- Basic → 1 month
- Pro → 6 months

----------------------------------------

UPGRADE TRIGGERS:

Whenever user hits a locked feature:

Show message:

"Unlock deeper insights and complete analysis with Pro."

For Ask limit:

"You’ve reached your daily limit. Upgrade to Pro for unlimited questions."

----------------------------------------

UI REQUIREMENTS:

- Clearly show "Basic" and "Pro" tags
- Highlight Pro as:
  "🔥 Most Popular"

- Show comparison (Basic vs Pro)
- Add strong CTA button:

[ Upgrade to Pro 🔓 ]

----------------------------------------

IMPORTANT RULES:

- Do NOT make Basic and Pro similar
- Pro must feel significantly more powerful
- Keep Basic useful but incomplete
- Encourage upgrade through curiosity, not force

----------------------------------------

FINAL RESULT:

- Clear user journey
- Strong monetization system
- High conversion potential
