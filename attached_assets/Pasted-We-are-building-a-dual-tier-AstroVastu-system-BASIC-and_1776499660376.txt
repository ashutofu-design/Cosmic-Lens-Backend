-------------------------------------
🟢 BASIC TIER — SYSTEM DESIGN
-------------------------------------

Goal:
Fast, simple, low-cost, addictive entry experience.
No image processing. No heavy logic.

INPUT:
- User kundli data (minimum):
  - Lagna (required)
  - Rashi (required)
- Optional:
  - Planet strengths (if available)

PROCESS LOGIC:

Step 1: Identify Lagna element and ruling planet
- Map Lagna → element (fire/earth/air/water)
- Map Lagna → favorable directions

Step 2: Apply simple Rashi support
- Check if Moon sign supports or weakens Lagna direction
- Adjust recommendation slightly (no deep logic)

Step 3: Match with generic vastu rules
- Example:
  Kitchen → SE ideal
  Bedroom → SW ideal
  Pooja → NE ideal

Step 4: Combine result
- Output:
  - “Ideal (aligned)”
  - “Acceptable (minor adjustment needed)”
  - “Not recommended”

Step 5: Suggest 1–2 simple remedies
- Color
- Direction facing
- Small object placement

OUTPUT FORMAT:
- Plain text only
- Max 80–120 words
- No sections
- No “scan” language

Example Output:
Your Lagna is Gemini. Air element supports West and North directions.
Kitchen in West is acceptable. Slight imbalance in fire element.
Suggestion: Add green element or plant near North wall.

STRICT RULES:
- Do NOT use advanced astrology (no Mahadasha)
- Do NOT simulate scan
- Keep logic deterministic and simple
- Keep cost minimal
